# Task Master API

Бэкенд-сервис для управления задачами, построенный на современных инструментах разработки Python. Проект демонстрирует навыки построения RESTful API, контейнеризации и автоматизированного тестирования.

## Технологический стек

* **Язык:** Python 3.12+
* **Фреймворк:** FastAPI (или Django DRF - укажи свой вариант)
* **Управление зависимостями:** uv
* **База данных:** PostgreSQL
* **Контейнеризация:** Docker, Docker Compose
* **Тестирование:** Pytest

## Основные возможности

* Полный цикл CRUD для задач (создание, чтение, обновление, удаление).
* Асинхронная обработка запросов.
* Валидация входящих данных.
* Контейнеризированная среда разработки.

## Быстрый старт с Docker

Для запуска проекта вам потребуется установленный Docker и Docker Compose.

1. Клонируйте репозиторий:
   git clone https://github.com/mauu1619/task-master.git
   cd task-master

2. Создайте файл .env на основе примера:
   cp .env.example .env

3. Запустите проект:
   docker-compose up --build

Приложение будет доступно по адресу: http://localhost:8000
Документация API (Swagger): http://localhost:8000/docs

## Локальная разработка

Если вы предпочитаете запускать проект без Docker, рекомендуется использовать менеджер пакетов uv.

1. Установите зависимости:
   uv sync

2. Запустите миграции базы данных:
   uv run alembic upgrade head  # Для Alembic
   # или используйте ваш инструмент миграций для FastAPI

3. Запустите сервер разработки:
   uv run uvicorn main:app --reload

## Тестирование

Для запуска набора тестов выполните команду:

uv run pytest

Тесты покрывают основные эндпоинты API и логику обработки данных.

## Структура проекта

* app/ — основной код приложения.
* tests/ — тесты Pytest.
* docker-compose.yml — конфигурация сервисов (API, DB).
* Dockerfile — инструкция сборки образа приложения.
* pyproject.toml — описание проекта и зависимостей для uv.
